# Usagi-Explode-Public

制作期間　2023年12月26日〜2024年1月7日  
製作人数　1人  

スコアを競う1人用3Dサバイバルシューティングゲームです。  
敵が迫り来る緊張感と爆弾で敵を倒す爽快感がコンセプトです。

Authentication 3.2.0
